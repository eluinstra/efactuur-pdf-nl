This zip-file provides two examples of Invoice messages.

To ensure that the examples are placed in the correct folder, 
this zip-file should should be extracted to the same folder as the 
zip-file containing the HR-XML 2.5 schema's has been extracted to. 
After unzipping, the examples can be found in the folder 
HRXML200704\HR-XML-2_5\SIDES\NL.
